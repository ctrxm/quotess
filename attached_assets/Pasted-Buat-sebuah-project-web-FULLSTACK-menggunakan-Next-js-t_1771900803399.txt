Buat sebuah project web FULLSTACK menggunakan Next.js terbaru (App Router) dengan TypeScript dan Tailwind CSS.
Tujuan project: website share quotes berbahasa Indonesia yang ramai/viral, dengan tampilan NEOBRUTALISM, terintegrasi Supabase, dan siap deploy ke Vercel.

==================================================
1) TECH STACK & KONFIGURASI DASAR
==================================================
- Framework: Next.js App Router + TypeScript
- Styling: Tailwind CSS
- Database: Supabase Postgres
- Library DB: @supabase/supabase-js
- OG Image: @vercel/og
- Target deploy: Vercel

Gunakan environment variables:
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY (OPTIONAL, server-only)
- ADMIN_SECRET (optional untuk admin panel)

Jika env Supabase belum diisi, aplikasi harus tetap jalan dengan dummy data dan menampilkan warning kecil di UI.

==================================================
4) DATA MODEL (SUPABASE)
==================================================
Buat file /supabase/schema.sql dengan tabel:

quotes:
- id uuid PK default gen_random_uuid()
- text text not null
- author text
- mood text not null
- status text default 'approved'
- created_at timestamptz default now()

tags:
- id uuid PK
- name text unique
- slug text unique

quote_tags:
- quote_id uuid FK ‚Üí quotes.id
- tag_id uuid FK ‚Üí tags.id
- primary key (quote_id, tag_id)

Tambahkan index untuk:
- quotes(status, created_at desc)
- quotes(mood)

Buat /supabase/seed.sql:
- 30+ quotes bahasa Indonesia
- Mood: galau, semangat, sindir, healing, kerja, cinta
- 10‚Äì15 tags populer Indo (moveon, produktif, sukses, patahhati, dll)

==================================================
5) SUPABASE CLIENT

Buat:
- lib/supabase/client.ts (client-side)
- lib/supabase/server.ts (server-side)

Gunakan anon key untuk read & submit quote.
Service role key hanya dipakai di server routes jika tersedia.

==================================================
6) API ROUTES
==================================================
Buat API di App Router:

GET /api/quotes
- filter: mood, tag
- pagination
- hanya status='approved'

GET /api/quotes/search?q=
- search di text & author (ILIKE)

POST /api/quotes
- submit quote UGC
- status default 'pending'
- auto-create tags
- rate limit IP sederhana (max 5 submit / 10 menit)

PATCH /api/admin/quotes
- approve / reject
- hanya jalan jika ADMIN_SECRET benar
- jika service role key tidak ada, tampilkan error aman
